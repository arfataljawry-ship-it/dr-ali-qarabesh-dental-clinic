## ✨ المميزات

- **إدارة كاملة للمرضى:** إضافة، تعديل، وحذف بيانات أي مريض بسهولة.
- **البحث الفوري:** البحث باسم المريض أو رقم الهاتف لحظياً.
- **حساب المتبقي تلقائياً:** المتبقي = المبلغ كامل − المدفوع، ويتحدّث أثناء الكتابة.
- **حفظ دائم للبيانات:** تبقى البيانات محفوظة بعد إغلاق المتصفح عبر `LocalStorage`.
- **تنبيه المواعيد:** فحص تلقائي لمواعيد الجلسة الثانية مع تنبيه قبل الموعد بيوم وفي يومه.
- **تلوين الصفوف:** موعد اليوم باللون الأحمر، وموعد الغد باللون البرتقالي داخل الجدول.
- **تنبيه صوتي:** نغمة تنبيه مولّدة برمجياً (بدون ملفات صوت خارجية).
- **لوحة إحصائيات:** إجمالي المرضى، مواعيد اليوم، مواعيد الغد، وإجمالي المبالغ المتبقية.
- **واجهة عربية احترافية:** تصميم عيادي نظيف يدعم الكتابة من اليمين إلى اليسار، ومتجاوب مع الجوال.

## 🚀 التشغيل المحلي

لا يحتاج المشروع إلى أي تثبيت أو خادم. يكفي:

1. تنزيل المشروع أو استنساخه:
   ```bash
   git clone https://github.com/USERNAME/dr-ali-qarabesh-dental-clinic.git
   ```
2. فتح ملف **`index.html`** بأي متصفح حديث (Chrome / Edge / Firefox).

> 💡 التنبيه الصوتي عند الإقلاع قد يتطلّب نقرة واحدة داخل الصفحة أول مرة، لأن المتصفحات تمنع تشغيل الصوت قبل أي تفاعل. زر الجرس 🔔 يعالج ذلك.

## 🌐 النشر على GitHub Pages

المشروع جاهز للعمل مباشرة على GitHub Pages لأن `index.html` موجود في جذر المستودع:

1. ارفع ملفات المشروع إلى مستودع جديد على GitHub.
2. من المستودع، افتح **Settings → Pages**.
3. في قسم **Build and deployment**، اختر:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` ثم المجلد `/ (root)`
4. اضغط **Save**، وانتظر دقائق حتى يصبح الموقع متاحاً على الرابط:
   ```
   https://USERNAME.github.io/dr-ali-qarabesh-dental-clinic/
   ```
